Наш магазин
Простой веб‑магазин с переключением темы и корзиной покупок.

 Описание
Это статическая веб‑страница интернет‑магазина с базовым функционалом:
отображение карточек товаров (пицца, наушники, книга, игрушка);
переключение между светлой (жёлтой) и тёмной (синей) темами;
добавление и удаление товаров из корзины с визуальной обратной связью;
счётчик количества товаров в корзине;
кнопки для перехода на страницы отдельных товаров и в корзину.

 Функционал
Переключение темы
Кнопка «Синяя тема» / «Жёлтая тема» переключает цветовую схему страницы.
При активации тёмной темы фон страницы и карточки товаров меняют цвет, текст становится белым.

Корзина покупок

Кнопка «В корзину» добавляет товар в корзину. При нажатии текст меняется на **«Добавлено»**, кнопка меняет цвет.
Повторное нажатие удаляет товар из корзины: кнопка возвращает исходный вид, счётчик уменьшается.
Счётчик в верхней части страницы показывает текущее количество товаров в корзине (иконка корзины + число).
Отдельная кнопка «Корзина» ведёт на страницу корзины (ссылка `index3.html`).
  
 Навигация
 У каждого товара есть кнопка «перейти на товар», которая открывает страницу с подробной информацией о нём (ссылки `index6.html`, `index7.html`, `index8.html`).

 Структура кода
HTML (`index.html`)
Базовая разметка страницы с заголовком, кнопкой переключения темы, счётчиком корзины и контейнером для карточек товаров.
Карточки товаров содержат: изображение, название, цену, кнопки «В корзину» и «перейти на товар».

CSS
Стили для светлой и тёмной тем (классы `body` и `body.dark`).
Стили карточек товаров, кнопок, счётчика и других элементов.
Эффекты при наведении (тени, анимация кнопок, подъём карточки).
Адаптивная вёрстка: карточки располагаются в ряд, при нехватке места переносятся на новую строку.

JavaScript

 Функция `toggleTheme()`:
    переключает класс `dark` у элемента `body`;
    меняет текст кнопки в зависимости от текущей темы.
Функция `buy(btn)`:
    отслеживает нажатие на кнопку «В корзину»;
    изменяет текст и стиль кнопки при добавлении/удалении товара;
    обновляет счётчик товаров в корзине.

 Запуск проекта
1. Сохраните код в файл `index.html`.
2. Поместите изображения товаров (`пицца1.jpg`, `наушник1.jpg`, `книга1.jpg`, `игрушка1.jpg`) в ту же папку, что и `index.html`.
3. Откройте файл `index.html` в любом современном веб‑браузере (Chrome, Firefox, Safari, Edge и т. д.).
 
Зависимости
Не требуются. Проект использует только HTML, CSS и нативный JavaScript.

Примечания
Изображения товаров.В коде указаны локальные пути к изображениям. Для работы в интернете загрузите изображения на сервер и обновите пути в атрибутах `src`.
Страницы товаров и корзины.Ссылки на `index3.html`, `index6.html` и т. д. являются заглушками. Для полноценной работы создайте соответствующие HTML‑страницы.
Корзина. Текущая реализация хранит количество товаров только в памяти браузера (переменная `count`). При обновлении страницы счётчик сбрасывается. Для сохранения данных потребуется серверная часть или `localStorage`.
